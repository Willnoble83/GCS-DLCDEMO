Google Cloud Storage DLC Demo
==============================
Unity Version: 2021.3.11f1
==============================

TO USE
-------------
1) Open The project in Unity
2) Load MainScene if it is not automatically loaded (It can be found in Assets/Scenes/MainScenes)
3) Hit play and click the buttons

To Enable DLC2
1) Navigate to the root folder of the project
2) Open the DLC Folder
3) Rename 'dlc2blocked' to 'dlc2'
4) Stop the play session if you have one running and restart it.
